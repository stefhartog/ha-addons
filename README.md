# Home Assistant Add-ons

This repository contains add-ons for Home Assistant.

## Featured add-on: Snoozefest

Snoozefest is a local-first alarm and timer brain for Home Assistant.

It gives you:

- reliable scheduling with persistent local state
- native MQTT discovery entities for manager, alarms, and timers
- optional dashboard UI cards for polished mobile control
- optional voice flows (deterministic + LLM wrappers)

Start simple with just the add-on, then layer on UI/voice when you are ready.

## Add this repository

1. In Home Assistant, go to Settings -> System -> Add-ons -> Repositories.
2. Add: https://github.com/stefhartog/ha-addons
3. Refresh and search for Snoozefest.

## Install Snoozefest

1. Open Snoozefest.
2. Click Install.
3. Start the add-on and check logs.
