# Home Assistant Add-ons Repository

## Supported Add-ons

- **Snoozefest** - Local alarm and timer service with MQTT discovery

## Installation

1. In Home Assistant, go to **Settings → System → Add-ons → Repositories**
2. Paste: https://github.com/stefhartog/ha-addons
3. Click **Create**
4. Refresh, search "Snoozefest", and Install
